##### 关于In、Out、Inout的说明：  
*	ease `in` :  加速度缓动
*	ease `out`:  减速度缓动
*	ease `InOut`:先加速度至50%，再减速度完成动画  
  
##### 方法介绍  
*	Quad：二次方缓动
*	Cubic：三次方缓动
*	Quart：四次方缓动
*	Quint：五次方缓动
*	Sine：正弦曲线缓动
*	Expo：指数曲线缓动
*	Circ：圆形曲线的缓动
*	Elastic：指数衰减的正弦曲线缓动
*	Back：超过范围的三次方缓动
*	Bounce：指数衰减的反弹缓动
